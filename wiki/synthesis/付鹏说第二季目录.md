## 季度概览

**时间范围**：2021年7月 — 2022年8月（约14个月）

**集数统计**：正式编号集数01–35（其中16、17、18期未发布或缺失），加上3集年终回顾（2021年年终回顾上/中/下）和1集加餐（黄金商品远期），共计约35集正式内容。

**时代背景**：
- 全球疫情后经济恢复期，主要经济体货币政策分化加剧
- 2021年下半年：全球供应链危机、能源危机（煤电、天然气、油价三线爆发）
- 2021年四季度至2022年：通胀压力超预期，美联储被迫从"暂时性通胀"转向激进加息
- 2022年2月24日俄乌冲突爆发，进一步加剧能源/商品系统性风险
- 上海疫情（2022年4月）叠加中美利差倒挂，国内市场面临双重压力

**核心主题**：汇率赛马机制（各央行退出节奏差异）、疫情后供给定价逻辑、能源危机本质、美联储从"主动回归正常"到"被迫激进加息"的路径转变、俄乌冲突下的系统性风险、中国实际利率测算与房地产库存、新兴市场风险预警

## 核心观点汇总

### 1. 美元指数框架重塑（欧洲日本化论断）
2012年欧债危机后欧洲央行"日本化"，欧元权重（约60%）降为被动变量，美元指数由单边美联储政策主导。实际上限90，向上破100的概率大于向下破90。经典表述：**"如果美国差，欧洲一定比美国更差，所以美元指数不会回60。"**

### 2. 汇率赛马框架（2021年疫情后退出周期）
各央行退出政策的排序（强→弱）：新西兰 > 加拿大 ≈ 美国 > 英国 > 澳大利亚 > 欧洲。核心工具：OIS（隔夜指数掉期）曲线发散/收敛。关键判断：**汇率升值≠央行收紧，升值与否取决于美国利率在序列中的相对位置。**

### 3. 供给定价为核心（疫情后商品）
需求端恢复是"既定的"（可预测），价格核心变量是供给端弹性。以原油为例：疫情后OPEC减产 + 沙特额外减产叠加需求曲线外扩 = 价格二次上冲。关键论断：**"供需曲线中，需求故事无穷无尽，但定价核心仍然是供给。"**

### 4. 美联储从"主动"到"不得不"
2021年11月为分水岭：之前是主动回归常态，之后变成被迫解决通胀问题。结果：被迫激进加息必然导致利率曲线快速倒挂，隐含衰退风险。**核心两难：解决通胀就得接受衰退风险；容忍通胀则失去央行信用。**

### 5. 中国实际利率测算的特殊性
中国不能用CPI替代资产端回报率，应用**房价同比变动**代替CPI，因为中国居民核心资产是房地产。民间借贷利率 < 房地产回报率：过去十几年"只要能借钱，投资就划算"。但一旦房价转向，实际利率从正转负，整个资产回报体系面临根本性重估。

### 6. 中美利差倒挂逻辑
中债收益率 = 股市权重板块（价值/银行/地产）的隐含回报率代理变量。中美利差倒挂意味着：资产端回报（中债）下降 + 负债端成本（美债）上升，双杀结构。2022年倒挂历史罕见，除2018年短暂倒挂外基本没有前例。

### 7. 俄乌冲突下的系统性风险
能源是俄罗斯的"经济核武器"，不是简单的供给短缺，而是引发相关领域市场系统性风险。LME镍市场为典型案例：流动性下降后潜在风险飙升，空头被多头逼仓，LME被迫暂停交易。关键判断：**"流动性一旦下降，潜在风险大幅增加——不需要有人刻意逼仓，自行调仓都可能引发剧烈波动。"**

### 8. 新兴市场风险预警（2022年）
核心观察指标：亚洲主要经济体外汇储备增速（除中日）。当增速从15%下降时，历史上对应1997年亚洲金融危机级别的风险。重点关注韩国：跟随美联储加息 → 本币资金成本拾升 → 资产价格承压 → 居民高杠杆风险暴露；相比之下，日本选择YCC（收益率曲线控制）是更合理的政策路径。

### 9. 黄金实际利率曲线结构性解读
疫情后黄金1740-1940美元区间宽幅震荡的原因：实际利率曲线发生**结构性变化**（短端通胀极高/名义利率极低，长端已开始抬升），加权后曲线中枢并未移动，因此金价只是波动而未突破区间。打破区间的条件：短端名义利率加速追赶短端通胀 → 金价破下限；或长端实际利率继续下行 → 金价破上限。

### 10. 铜的完整转身路径（2022）
宏观层面：G2 PMI综合（中美PMI均已转弱）+ 全球权益市场下行。微观层面：铜精矿加工费（TC/RC）跌破盈亏平衡线，冶炼厂亏损，供给端开始被动减少。时间节点：2021年Q1中国新兴市场基金开始分化（先行指标），2022年Q1全球权益下行补充确认（滞后指标）。

## 分析框架（第二季新增/强化）

### 框架一：汇率赛马中的"速率竞争"
将各国央行退出政策制作成排序表，核心三维度：
1. 购债/扩表节奏（减速/停止/缩减）
2. 加息时间表和幅度预期（OIS曲线定价）
3. 对通胀的官方表态（是否"暂时性"）

**应用规则**：强弱序列中越靠前的货币偏强，越靠后的货币偏弱。汇率方向=两国在序列中的相对位置变化。

### 框架二：交易员判断转折点的核心变量法
非所有因子同时翻转，只有**核心因子**先发生变化。识别核心因子 vs. 噪音因子的关键：看是否改变了主要逻辑。类比：价格顺势期各方信息一致，转折期信息开始分化，此时捕捉核心矛盾变化是交易员素养的核心。

### 框架三：期货 vs. 股票选择框架（以生猪为例）
商品价格触底后选股票而非期货的逻辑：
- 近月期货：价格在底部时时间成本极高（Super Contango结构）
- 远月期货：流动性差，价格已经包含了基本面恢复预期
- 股票：无时间成本，最差时候买入，体现"最坏时候=买入时机"

### 框架四：两油价差宏观含义
WTI vs. 布伦特价差扩大 → 市场在交易美国经济放缓先行预期：
- 布伦特由OPEC+定价，更接近利率短端（通胀逻辑）
- WTI由市场化定价，供给弹性更强，更早反映经济需求放缓
- 利率曲线扁平化 = 两油价差扩大，二者逻辑底层相通

### 框架五：香港M2作为全球流动性代理变量
香港联系汇率制度 → M2增速 = 全球流动性的变化 × 资金成本（实际利率）双变量：
- 总量（香港M2）反映流动性总量
- 成本（美元实际利率）反映估值上限
- 应用：分析恒生科技 vs. 恒生指数分化的根本原因

## 按主题分类的精华

### 主题一：美联储/利率政策

**【11】胀的预期与美联储的步伐**
- 2021年三季度能源危机推高通胀预期（绳子的通胀端被拉），同时美联储加息预期（名义利率端）也在跟进
- 两根绳子相互拮抗 → 黄金在1800附近低波动区间震荡
- 前瞻性判断：若能源危机解决，通胀预期回落快于名义利率抬升 → 实际利率加速上行 → 黄金下行

**【23】利率曲线走势与美联储的迫切性**
- 2021年11月转折点：美联储从"主动回归"变为"不得不"
- 不得不的代价：快速加息 → 利率曲线快速倒挂 → 衰退风险
- 美联储真正能做的：打掉支撑资产价格的流动性；不能做的：解决供应链、能源结构性问题

**【26】俄乌冲突下美联储的选择**
- 俄乌危机同时作用于通胀（更高）和增长（更低），美联储面临两难
- 鲍威尔态度：优先解决通胀失控风险，即使经济放缓也要坚持加息
- 关键表态：中性利率约2-2.5%，不排除单次加息50BP

**【28】耶伦vs鲍威尔两次加息缩表周期对比**
- 耶伦时代加息初期（2016-2018）：经济从低位修复，香港M2增速上升，流动性宽裕
- 鲍威尔时代（2022）：缺少"加息初期良好"阶段，直接从疫情超低利率跳至快速加息
- 两次周期的本质差异：耶伦是"高位扁平化"，鲍威尔是"低位扁平化"

**【25】从利率看权益分子分母**
- 资产回报率（分子）与资金成本（分母）的4种组合
- 2014-2018耶伦时代：分子（资产回报）> 分母（利率）→ 最佳状态
- 2022：分子下降 + 分母抬升 → 最差状态（衰退交易）

### 主题二：中国宏观

**【19】房地产库存数据解析**
- 关键定义区分：新开工面积（不含跨期复工）、施工面积（含所有在建）、竣工面积（已完工可交付）、待售面积（现房，不含期房预售）
- 待售面积上升 = 现房销售 < 竣工面积（现房去化慢）
- 开发商在房价上涨周期刻意拖延工期的行为逻辑

**【21】2022年A股展望：实际利率变化下的风格切换**
- A股传统行业权重与全球通胀预期高度正相关（中国是全球制造加工环节）
- 低实际利率 → 成长/赛道股占优；实际利率上升 → 价值/低估值板块相对占优
- 2022年判断：实际利率开始从历史低位回升（2021年四季度已启动），赛道股高估值风险上升

**【29】中美利差倒挂含义**
- 中债收益率 ≈ 股市价值板块（权重股）的隐含回报代理
- 美债收益率 ≈ 全球资产负债端成本代理
- 中债下行 + 美债上行 = 资产回报下降 + 融资成本上升 = 双杀结构
- 历史稀缺性：2022年中美利差倒挂幅度超过历史大部分时期

**【31】中国实际利率如何测算**
- 中国不能用CPI做实际利率分子（CPI未包含房地产资产回报）
- 正确测算：资产端用房价同比，负债端用民间借贷利率或官方名义利率
- 历史状态：民间借贷利率低于房地产回报率 → 实际利率大部分为负 → 加杠杆/借钱投资房地产划算
- 含义：房价一旦转向，实际利率翻正，整个资产回报体系面临重估

**【20】出口与人民币**
- 2015年"8·11"汇改后人民币主要受贸易（经常项目）驱动
- 疫情后出口异常强劲（全球供需错配下中国承接全球总需求镜像）
- 预判（2022年）：当全球供应链恢复、西方制造业产能回归，中国出口顺差将收缩，人民币面临从升值转贬值的压力；6.4的中间价可能难以维持

### 主题三：大宗商品

**【4】需求恢复确定，定价核心是供给（原油）**
- 疫情后需求曲线外扩是确定的，供给曲线弹性决定价格上涨幅度
- OPEC在2020年五六月份1000万桶/天减产后油价企稳，再叠加沙特额外减产 → 价格二次上冲
- 未来定价核心：OPEC是否改变供给曲线（减产/增产）

**【7】CFTC原油持仓与交易对手关系**
- CFTC持仓数据是观察市场结构的核心工具
- 非商业多头（投机）与商业空头（套保）之间的博弈关系
- 通过观察持仓结构判断市场是否具备逼仓条件

**【12】利用能源危机的超级挤仓**
- 2021年三季度LME铜0-3价差最高超过1000美元（有数据以来历史最高），但绝对价格并未创新高
- 这是典型的"挤某些头寸而非挤整个市场"特征
- 操作逻辑：利用基本面窗口期（低库存/能源危机），在不导致绝对价格剧烈波动的情况下挤压特定空头

**【30】铜的完整转身路径**
- 宏观指标：G2 PMI均已转弱，全球权益指数下行
- 微观指标：铜精矿TC/RC价格持续跌破盈亏平衡线，冶炼厂亏损
- 时间节点：2021年Q1新兴市场基金先行分化（铜的顶部），2022年Q1全球权益确认（供给端反应滞后）

**【9】多PPI空CPI反转来了？**
- 2021年一个主要交易主题：多供应端资产（PPI/大宗），空需求端资产（CPI/消费品）
- 反转信号：猪肉价格触底，微观上空生猪等CPI资产已出现政策性干预
- 宏观信号：PPI相关资产开始回落，CPI相关资产开始抬升

**【15】特朗普vs拜登如何干预油价**
- 两届政府能源安全战略一致：油价超过70-80美元/桶就会干预
- 特朗普2018年和拜登2021年的干预路径、时点、价格区间几乎相同
- 区分：能源发展战略（两党不同）vs. 能源安全战略（两党一致）
- 50-70美元/桶是OPEC+与消费国的利益均衡区间

### 主题四：汇率外汇

**【1】权重对美元指数的影响**
- 欧元权重约60%，日元约14%；美元指数实质是美欧的相对关系
- 欧洲央行2012年后日本化 → 美元指数新均衡区间90-100
- 判断方向：向上（>100）概率大于向下（<90），因为欧洲永远比美国更差

**【2】【3】汇率赛马机制（上下）**
- 核心工具：OIS隔夜指数掉期曲线（观察各国央行加息预期）
- 2021年序列：新西兰 > 加拿大 ≈ 英国 > 澳大利亚 > 欧洲
- 美国位置决定方向：美国在序列中越靠前，与排名靠后的货币的汇率越强

**【10】流动性看中美欧日货币信贷脉冲**
- G4央行资产负债表扩张对全球权益回报率的领先效应（约6个月）
- 2014年后G2（中美）主导全球流动性增量，欧日被动跟随
- 每次危机后的货币信贷脉冲越大，潮水退去后的结构失衡越严重

**【8】韩国出口数据**
- 韩国出口数据是全球总需求的高频"金丝雀"指标
- 2021年韩元/人民币走势出现劈叉：韩元已率先转弱
- 含义：韩国出口领先人民币出口下行约一个季度

### 主题五：市场事件与波动

**【6】存量资金下的结构性行情**
- A股在缺乏增量资金时，只能通过内部轮动形成结构性行情
- 增量资金（流动性总量）vs. 结构性轮动（资金在板块间转移）

**【13】如果猪肉价格触底了：股票 vs. 期货**
- Super Contango结构下：时间成本极高，做近月期货消耗严重
- 基本面触底建仓应优先股票，而非期货
- 商品期货逻辑：最差时候=最好买入时机（基本面开始出清）

**【14】交易员思维：判断转折的核心变量**
- 信息分层：价格顺势时信息一致，转折时信息开始分化
- 核心因子翻转 vs. 噪音因子翻转：判断是否改变了主要逻辑
- 案例：2012年澳元，交易澳元下跌逻辑的时间轴与因子演化

**【24】交易员如何应对突发公共事件**
- 公共性事件的不确定性 → 唯一可交易的是波动率
- "买预期卖事实"经典演绎：2022年2月24日俄乌开战当日VIX冲至40，收盘回落至27
- 判断波动率是否已经过高定价：超高VIX + Backwardation结构 = 不确定性已充分定价

**【27】俄乌冲突下的系统性风险**
- LME镍市场案例：制裁俄罗斯 → 流动性骤降 → 潜在风险飙升 → LME暂停交易
- 系统性风险不需要主动逼仓触发，流动性极低时自行调仓即可引发
- 俄罗斯能源是"经济核武器"，不只是供给短缺问题

**【32】新兴市场风险明牌**
- 2022年核心外部压力：美联储激进加息 → 全球美元负债成本飙升
- 新兴市场两难：跟随加息则本国资产价格压力大；不跟随则汇率贬值
- 危险信号：亚洲主要经济体（除中日）外汇储备增速从15%快速下降，历史上对应1997年级别风险

**【33】韩国弱于日本（股债汇三市场）**
- 韩国选择跟随美联储加息：债券收益率+近200BP，但韩元与日元走势几乎一致（都在贬值）
- 日本选择YCC（收益率曲线控制）：利率被锁定，日元贬值但资产价格未受国内利率冲击
- 付鹏判断：日本央行的YCC更合理，韩国是股债汇三杀，日本至少只有汇率承压

**【34】黄金跟踪：实际利率曲线**
- 2022年黄金长期维持1740-1940美元区间的深层原因：实际利率曲线结构扭曲（短端实际利率仍低，长端已上行），加权中枢未变
- 历史上从未出现的情形：利率曲线结构扭曲如此极端（2-3年期内通胀极高而名义利率极低）
- 打破区间的两个路径：①短端名义利率快速追赶通胀 → 金价破下限；②长端实际利率继续下行 → 金价破上限

**【35】两油价差走扩的宏观含义**
- WTI vs. 布伦特价差扩大（WTI走弱）= 美国需求放缓预期 + 市场化供给弹性更强
- 布伦特由OPEC+（非市场化力量）定价，价格更有支撑；WTI由市场化定价，供给弹性对价格影响直接
- 两油价差扩大 ↔ 利率曲线扁平化（利率曲线走向倒挂）在底层逻辑上高度吻合
