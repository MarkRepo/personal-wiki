## 核心结论

《大类资产框架手册》是付鹏2019年在华尔街见闻录制的系统性大师课，共35节，覆盖从经济世界观构建到各大类资产（商品、汇率、证券）分析逻辑的完整框架。课程的核心思想是：

1. **自上而下的宏观框架**：用"分工与分配—债务与杠杆—收入与贫富"三组关键词理解全球经济的底层逻辑，再向下传导至大类资产的定价
2. **打通宏观与交易的链条**：宏观经济学家讨论"可能产生的结果"，交易员需要的是中间的传导路径，两者之间的连接才是核心价值
3. **大类资产的精确界定**：不是所有资产都适合宏观框架，只有流动性强、纵深够深、金融属性主导的资产才属于大类资产（FICC范畴）
4. **利率曲线是核心**：所有大类资产的联动都以利率曲线为枢纽

（来源：付鹏大师课·大类资产框架手册，2019年6月，华尔街见闻）

### 二、什么是大类资产——付鹏的精确界定（1.1节）

付鹏对大类资产的界定有严格标准，不是"所有资产都可以放进宏观框架"：

**纳入大类资产框架的判断标准：**
- 流动性足够：T+0 或接近即时，纵深够深
- 金融属性主导（而非基本商品属性主导）
- 参与者结构成熟，受宏观变量驱动

**大类资产分类（FICC框架）：**

| 资产类别 | 主要品种 | 配置/关注方式 |
|----------|----------|---------------|
| **债券**（固定收益）| 美国、欧洲、日本、中国、澳洲 | ETF + 衍生品 |
| **权益**（股票）| 美国、欧洲、日本、香港、中国（龙头ETF）| 不选个股，用ETF/指数配置 |
| **汇率** | G7汇率为主；韩元、台币等为关注序列 | 交易G7；新兴市场汇率寻替代资产 |
| **商品（大类）** | 原油（利率型变量）、铜（宏观景气指标）| 期货、ETF |
| **贵金属** | 黄金（按债券逻辑）、白银（按商品逻辑）| 注意黄金≠白银的定价逻辑差异 |

**明确排除在框架外的资产：**
- 农产品（红枣、苹果期货等）：纯商品属性，微观因素主导
- 小众金属（钴、锂）：小众资产逻辑，需微观专项研究
- 新西兰、土耳其、巴西等小市场股票：流动性/纵深不够
- 铁矿石、煤炭：属于观察/研究序列

**黄金 vs 白银的关键区分：**
黄金按债券逻辑定价（实际利率的倒数），白银有大量工业用途，按商品逻辑定价。2011-2012年中资机构混淆两者逻辑，被华尔街"逮住"，损失惨重。

### 三、打通宏观与交易——付鹏的方法论（1.2节）

**核心洞察：宏观与交易各有其位**

- 宏观经济学家的职责：讨论"如果这样做，可能产生的结果是什么"——提供方向和框架，不提供具体交易结论
- 交易员的工作：把宏观矛盾层层传导，找到**中微观层面**的实际矛盾，由此构建交易架构

**最常见的错误模式：**

> 宏观大叙事（中东战争故事）→ 直接跳到具体价格结论（油价必涨到100）

这中间缺失了：市场结构分析、期货曲线结构、参与者仓位、即期供需动态等中间层次。付鹏批评这种"虎头蛇尾"的推导方式带有强烈赌性，不构成完整交易体系。

**正确的链条：**

```
宏观背景 → 政策方向判断 → 中观（行业/资产类别）分析 
→ 微观（市场结构/期货曲线/仓位/流动性）→ 交易决策
```

**两种极端主义都是错的：**
1. "宏观决定一切"——直接从宏观大结论推交易，跳过中微观
2. "宏观无用"——只看微观价格，忽视政策方向和系统性风险的防御意义

**宏观的实际价值：**
- 判断政策方向，提供战略定向
- 识别系统性风险点，构建防御
- 不是用来提供交易结论，而是优化中微观决策的上下文

### 四、经济世界观的三大核心问题（1.03-1.10节，OCR全文）

**三大核心问题（三组关键词）：**（来源：1.03节，2019-06-10）

1. **分工与分配** → 决定财富如何创造
   - 来源于《国富论》，延伸出全球经济一体化下国家间分工
   - 索罗斯1997年狙击香港，核心是利用国际分工和资本流动中的中微观矛盾
   
2. **债务与杠杆** → 决定财富如何扩张
   - 达利欧（《债务危机》）的核心研究领域
   - 财富扩张的来源：从劳动价值 → 债务价值 → 金融化（杠杆价值）三阶段

3. **收入与贫富（分配）** → 决定增长能否持续
   - 金融行业"一半天使一半恶魔"：创造财富的同时拉大贫富差距
   - 2008年金融危机"到现在还在延续"，正在影响"最害怕的问题——收入和贫富矛盾"

**经济增长模型（1.04-1.05节）：**

```
经济增长和效率 = f(债务/杠杆, 分工/分配)
              分子：债务和杠杆
              分母：分工和分配带来的收入
```

三阶段路径（以英国/中国为例）：
- **第一阶段**（劳动价值）：干实体，收入按斜率线性增长
- **第二阶段**（债务价值）：干房地产，负债产生更高收入，财富差距拉开
- **第三阶段**（金融化/杠杆价值）：干金融，最大限度利用杠杆，贫富差距达到极值

中国对应节点：2002年前干实体 → 2002年后干房地产 → 2008年后干金融

**应用到证券市场（1.05-1.06节）：**
```
股票价格 = f(债务/杠杆, 投机性波动, 收入/斜率)
```
- 成熟阶段市场（美国）：长期增长斜率高，以被动投资ETF为主
- 发展阶段市场（中国A股）：投机性波动占主导，关注换手和杠杆而非价值
- 付鹏判断：中国只有约150只股票具备长期价值投资属性，其余更多是交易博弈

**美国基金从业人员的四种类型（1.06节）：**
1. 活得足够长的：平滑高波动，展示10年平均回报
2. 低波动率做三五年：赚到分红退休然后写书演讲
3. 委托被动性投资：付管理费，自己关注防御风险
4. 危机一战成名者（如《大空头》）：赌对后必须立即解散，否则会叶回去

**债务不同于传统思路（1.07节）：**
- 债务/收入比不同国家处理方式完全不同，不能简单认为超过某比值就必然出问题
- 日本/欧洲三大政策（货币/财政/行政）都受限，中国和美国相对空间更多
- 2008年后杠杆从西方（伦敦/纽约）迁移到东方（上海/北京），金融从业者随之迁移

### 五、全球经济三级分工框架（1.08-1.10节）

**全球三级分工架构：**（来源：1.08节）
- **消费国**（第一集团）：发达国家，欧美，靠资本和债务扩张驱动需求
- **生产国**（第二集团）：中国、德国、日本、亚洲四小龙，依次承接制造业
- **原材料输出国**（第三集团）：澳大利亚、新西兰、资源国

债务分布差异：
- 消费国（美国）：居民部门债务为主
- 生产国（中国）：企业/生产部门债务为主
- 原材料国（澳大利亚）：生产+居民双重债务，正反馈时可以，负反馈时双杀

**三大失衡（1.09-1.10节）：**
1. **分工失衡**：发达国家把低端制造业全迁出，但底层工人无法转型，内部收入分配恶化（美国铁锈区、英国纽卡斯尔/谢菲尔德）
2. **消费国内部失衡**：分配自上而下，华尔街赚钱但底层工资不增；房价涨而工资不涨
3. **生产国与消费国的矛盾**：零和博弈——中国拿走的是别国核心收入，而非别人不要的部分

**全球经济一体化核心矛盾（1.10节）：**
- 消费国通过"自上而下"的税收/福利分配；生产国通过"自下而上"的税赋式分配
- 全球经济一体化已经结束，未来是"逆一体化/区域经济一体化"
- "日本化将蔓延到大部分国家"：欧洲、澳大利亚、新西兰、加拿大等
- 美元贬值是全球经济一体化正反馈时的自然结果（生产国利率高，资本双顺差流入）

**正反馈路径（付鹏描述）：**
```
消费国低利率 → 总需求扩张 
  → 生产国出口/生产过热 
    → 生产国利率/通胀上行 
      → 资本从实体流向虚拟资产（房产/金融） 
        → 生产国债务增加/贫富差距拉大 
          → 劳动力成本抬升 
            → 产业向更低成本国家转移（第二次分工）
```

### 六、大类资产的核心——美债（2.3-2.4节，OCR全文）

美债是所有全球大类资产的"锚"：

- 美债收益率（特别是10年期）是全球无风险利率的基准
- 从1980年到2019年，债务周期的末端，货币政策临近边际值
- **货币政策有效性正在大幅减弱**（2.3节核心观点）：各国收益率往1%、0方向走，已接近极值

**货币政策传导路径失效的根本原因：**
> "货币政策放水的目的是推动信用的扩张，但如果债务不出清，信用是扩不出来的。干放水，信用就是不扩张。"（来源：2.3节）

付鹏2019年预判：欧洲日本化、澳大利亚可能也日本化；所有国家最后都要靠行政手段或结构性改革，不能单靠货币政策。

**全球债券市场的carry trade本质（2.3节）：**
- 全球债券市场本质上是套息交易（carry trading）
- 日本投资者套利美国国债的收益率受Basis Swap和Libor走高影响

**曲线结构的交易逻辑（2.4节）：**
- 陡峭化（Steepening）→ "Buy the Curve"，买短期国债、卖长期国债期货
- 平坦化（Flattening）→ "Sell the Curve"，卖短期国债、买长期国债期货
- **TUT交易（Two-year Under Ten-year）**：美债2-10利差交易，使用CME网站提供的动态对冲比率（BPV/BPV）配置
- 日本东证（TOPIX）是美债长短端利差的最佳对标物：1990年后东证几乎无斜率，随长短端利差扩张/收缩波动

**实战策略：**
- 左手持有标普ETF（获取长期斜率收益），右手做空日本东证（对冲投机性波动风险）
- 这个组合在长周期下始终取得中间收益，既保留斜率，又对冲波动
- VIX深度Contango特性：可用小资金持续卖VIX，每年取出利润，3-4年必有一次爆仓，但总体正收益

**常用债券ETF（2.4节）：**
- TLT：美国国债20年以上ETF
- IEF：美国国债7-10年ETF
- LQD：投资级公司债ETF（600家流动性好的公司债）
- HYG：高收益债（垃圾债）ETF

### 七、全球大类资产反馈逻辑（2.1-2.2节）

**核心内核变量（2.1节）：**
- 各国管理层（货币/财政/行政政策管理层）+ 原油 → 生成所有大类资产的核心驱动
- 美元在全球经济一体化正反馈期必然贬值（生产国利率高于消费国）
- 高波动率时期：1997-2000年互联网泡沫、2008年金融危机、2009年欧债危机、2014-2015年中国债务问题

**波动率分析方法（2.2节）：**
- 使用波动率而非K线分析大类资产：低波动率=正反馈运行，高波动率=系统性风险
- 跨资产指标：**澳元/日元交叉汇率**——两个都是反映全球总需求的敏感货币，两者之比是全球风险情绪的优质代理变量
- 波动率和澳元/日元不同步时的处理：同步时大胆，不同步时小心
- 2014-2015年中国股市牛市：跨资产观察可以清晰看出是"水牛"（由杠杆保证金决定）

**所有汇率/商品/资产价格都是利率作用的结果，不是原因：**
> "因为美元涨了所以其他资产跌了"——这是把两个结果的因果关系搞反了，两者都是利率作用的结果

### 八、大宗商品分析——供需与时间价值（3.01-3.10节）

**金融杠杆化是大宗商品十年牛市的核心（3.01-3.02节）：**
- 2002年美国金融系统松绑，华尔街FICC部门全面介入大宗商品产业链（融资/仓储/贸易/生产）
- 三层需求并存：真实需求 + 投资性需求 + 投机性需求
- 2006年全球对能源真实总需求增速已经停滞，2006-2008年油价上涨更多是金融行为
- 2009-2012年，大量商品（铜/铝/铅/锌/大豆/棕榈油）承载融资套利功能：借美元→换人民币资产（中美利差+人民币单边升值3%→实际借贷成本为负）

**铜的特殊属性（3.02节）：**
- 2005年6月到2008年金融危机，铜价再未创新高（真实总需求推不动时商品就到顶了）
- 铜供应链短，供应周期短，对真实需求变化反应更敏感
- 铜存在大量"影子库存"（不体现在交易所库存中，用于质押融资）
- 利率抬升 → 仓储套利成本上升 → 影子库存变成显性库存 → 价格承压

**大宗商品分析的期货曲线结构（时间价值）：**
- **Contango（期货升水，远月溢价）**：库存充裕，供应宽松；持有者支付展期损失
- **Backwardation（期货贴水，近月溢价）**：现货紧缺，即期需求强；持有者获展期收益
- VIX本质上也是深度Contango，通过移仓不断消耗时间成本

**商品之王：原油的特殊地位（3.06节，OCR全文）：**
- 原油的分析原则："抛弃需求端，直接关注供应"
  - 需求端：BP养300多号研究员做年鉴，普通基金无法复制
  - 供应端：管道/油路/浮仓数据透明，可购买；地缘/OPEC等是公开新闻
- **油价双杀规律**：
  - 油价暴涨 → 消费国（发达经济体）风险（利率抬升，债务成本上升）
  - 油价暴跌 → 生产国（新兴市场）风险（企业/生产部门债务无法偿还）
- **油价对通胀的传导**：通胀/通缩 → 货币政策差异 → 资本流动 → 利差 → 大类资产

**原油供应端的囚徒困境（3.09节，OCR全文）：**
- OPEC本质是囚徒困境：集体减产才能维持高油价，但每个成员减产就丢市场份额
- 2014年油价暴跌：沙特主动测试，选择不减产反而增产，看页岩油弹性
- 结论：**OPEC干不过页岩油**——油价跌沙特必须减产，油价涨沙特才能增产；调节阀门在沙特
- 页岩油弹性：只有油价在极低价格维持3个月才会影响产出；短期油价波动对其扰动很低
- 付鹏2019年预判：未来5-10年OPEC定价能力极弱，美国（休斯敦+华盛顿）将是全球能源定价核心

### 九、汇率分析框架（4.1-4.2节，OCR全文）

**付鹏汇率决定体系（4.1节）：**
- 理论基础：国际收支学说（经常账户CA + 金融账户K）
- 补充理论：购买力平价、资本市场学说、行为人动学
- 方法论来源：与索罗斯《金融炼金术》框架"一模一样"——基于全球收支平衡考察资本流动

**汇率"好做"的原因（4.1节）：**
- 汇率不好做的根本原因是杠杆率太高（200倍），不是方向判断
- 把杠杆率压低到合理水平，基于中长周期框架，汇率投资极其稳定
- 央行数据一个月一次，汇率变化速度慢——至少应看周K/月K，不适合日内交易
- "汇率好做就是因为杠杆率低，把杠杆率压低，放到长周期，这个钱就属于你该赚的"

**核心方法：利差和利差预期（4.2节）：**
- 汇率分析第一步：国际收支（经常账户+资本账户历史分析）
- 汇率落地到交易时：直接关注货币政策结果，即利率展望和利率预期
- **高频数据优于低频数据**：不等央行声明，用OIS（隔夜指数掉期）期限结构提前感知降息预期
  - 短端OIS和长端OIS的差值扩大 → 市场已把降息预期计入
  - 案例：2012年澳元，当澳洲经济数据出来后OIS打破横盘区间，率先信号做空澳元，比央行声明早几个月

**付鹏人生第一桶金——2008年英镑（4.2节）：**
- 背景：美国已大规模降息，英国维持高利率（英格兰央行认为危机不会蔓延）
- 英镑在2.0附近横盘7个月
- 最终英格兰央行认错，一次性降息到位，利差补上，英镑历史性暴跌
- "30-40年唯一一次英镑最大的交易机会"，等待7个月才开始

**欧洲"双利差"问题（4.2节）：**
- 欧洲有两个利差风险：对内利差（欧债危机/成员国利差）+ 对外利差（vs美国）
- 2011-2013年交易的是对内利差（欧债危机）
- 2014-2016年交易的是德拉吉期权（ECB宽松导致对外利差走扩）
- 2017年欧元反弹：押注ECB退出，但债券交易者比汇率交易者早3-4个月发现ECB无法退出

**汇率分析不谈美元指数（4.1节）：**
> "加权指数是一个结果，不可能说因为加权指数，所以怎么样"——分解开看欧元/日元更有效

### 十、证券市场——美股与利率（5.1-5.2节，OCR全文）

**美股长周期结构（5.1/1.06节）：**
- 美股真正的长周期增长启动于1993年（非2008年后）
- 1980-1993年是里根政府结构性改革期
- 100年间仅两次长达10年的停滞（横盘），其余大部分时间都是长期增长
- 两次大波动原因：1997-2000年互联网估值泡沫；2008年金融危机（流动性风险，非估值问题）

**被动投资是美股最优策略（1.06节）：**
- "不是巴菲特成就了美国证券市场，是美国证券市场成就了长期价值投资"
- 1993年后斜率太高，巴菲特反而连续跑输大盘
- 美股：买篮子ETF长持，成本最低，不需要选股

**价值板块 vs 成长板块与利率的关系（5.2节，OCR全文）：**
- 成长板块（Duration长）：现金流在远期，对利率高度敏感
- 价值板块（利率代理）：用公共事业类板块（Utilities）代理利率资产，相当于"用股票表达债券"
- 关键比值：**信息技术/公共事业比** = 成长/利率比

**防御策略实例（5.2节）：**
- 2017年底付鹏开始加防御：左手仍持有ETF，右手加对冲
- 具体操作：
  - 卖成长板块ETF，买公共事业类ETF → 做对冲
  - 一旦美联储真的降息，平公共事业ETF，转单边空成长板块
  - 用成长/利率比值判断进出：比值回到6.5附近可加仓
- 2008年全年买任何资产都是划算的，因为"负债成本的下降速度更快"（基准利率降至0.25%）

### 十一、策略维度的思考（6.1-6.2节，OCR全文）

**"最伟大的交易员"定义（6.1节）：**
- 不是做对行情，而是"创造了行情、创造了规则、创造了价格运行机制"
- 案例：库欣仓储套利（BP交易员发现上下游价差，建仓库调剂，两亿美元三年赚回）
- 最伟大的是套利者：不预测方向，只吃确定性价差

**策略不存在圣杯（6.1节）：**
- 套利型、趋势型、开放式、家族式、封闭期差异——每种都有赚钱方式
- 对年轻人的建议：拼命挣工资，节省，找到机会满仓赌一把（因为年轻的优势是本金小、可以承担损失）
- 对有钱人的建议：低杠杆，赚固定利息性收入，两年找一个机会做一把

**资金属性和交易维度（6.2节）：**
- 第一步：确定资金属性（规模/背景/风格/目标年限）→ 确定交易维度（短期/中长期）
- 宏观大类资产交易天然是中长期交易
- 汇率：一周看一次，一个月过一次所有数据；不要频繁交易
- **不要陷入预测陷阱**：决定头寸了结的不是价格，而是"对应的相反事件开始出现"
- 对冲中的替代思路：错过股票行情？可以做国债期货对冲，比直接追指数波动率更低
- 未来的大类资产公司形态：不要自设交易员，而是配置不同特点的交易员/基金组合，自己做大类资产分配者
