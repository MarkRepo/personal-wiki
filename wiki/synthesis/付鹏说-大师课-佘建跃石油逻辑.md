## 课程概览

### 讲师背景

**佘建跃**：中海油（CNOOC）、中化（Sinochem）资深实货原油交易员，从业20余年。

- 1997年起在中化从事国际原油现货采购及炼厂/港务物流知识积累
- 2008年后加入中海油，主导国际原油实货贸易及套利/套保业务
- 期间经历多个油价大周期（2008年暴跌、2014年崩塌、2016年底部）
- 2012年受同事李卉（中化赴中海油）一份贸易模式PPT启发，系统整理价格关系与油价规律
- 著有《阳谋论》（机械工业出版社），由机工社编辑王峰约稿

本课程2019年在华尔街见闻「大师课」平台录制，共23讲（含前言），系统阐述从实货贸易视角看油价内在逻辑。佘建跃的核心判断：油价不是阴谋，而是可以从**产油国/消费国利益博弈（阳谋）**、**基本面三要素（供需存）**与**三类价差（跨区/裂解/跨期）**共同理解的体系。

## 核心分析框架

### 三类价差与基本面对应模型

佘建跃最核心的原创框架：**大宗商品跨区基本面与三类价差关系模型**。传统分析从数量（供应量、需求量、库存变化）切入，但预测精度极低（"分析师困境"——只能算日均速度，无法还原实时速度曲线）。更有效的方式是用**价格信号替代数量信号**，三类价差分别映射三个基本面要素：

| 价差类型 | 基本面映射 | 逻辑 |
|----------|------------|------|
| **跨区价差** | 边际供给（进出口） | 套利流动修正区域供需失衡；区间波动→各自平衡；单边突破→结构性变化 |
| **裂解价差** | 需求强度 | 成品油/原油价差反映炼厂盈利，进而驱动开工率；需求好→裂解价差扩大→原油需求增加 |
| **跨期价差（月差）** | 库存水平 | Contango→供大于求（正套困货有利）；Backwardation→供不应求（反套去库存快） |

**关键推论**：
- Contango深时，正套无风险套利出现，市场自发囤货，供大于求被库存吸纳；
- Backwardation时，反套释放库存极快，但库存不能为负——因此Backwardation结构下油价上涨动能更强、持续性更强；
- 正套与反套不对称：供大于求时库存可无限增加（受仓储约束）；供不应求时库存清零即止。

### 实货价格形成三步骤（期货→现货→定盘）

实货原油不是固定价，而是**公式定价（基差定价法）**：

1. **期货市场**（价格发现，绝对价格）：WTI / 布伦特 / DME阿曼期货确定绝对价格基准
2. **现货市场**（价格传递，相对价格）：通过普氏（Platts）、阿格斯（Argus）等评估机构的迪拜现货窗口，确定跨区品质差
3. **评估定盘**（价格确认）：普氏/阿格斯公布每日定盘价，用于实货结算

**实例**：沙特轻质油FOB价 = DME阿曼期货全月均价 & 普氏迪拜现货全月均价 的算术均值 + 沙特官方升贴水（OSP）

- **升贴水（Differential）**由两部分决定：品质差 + 时间价值
- 品质差来自裂解价差（各产品收率×裂解价差加权）；时间价值来自跨区平衡关系（EFS）

## 供需基本面

### 全球供需平衡表三大机构

| 机构 | 立场 | 特点 |
|------|------|------|
| **IEA**（国际能源署） | 消费国（OECD） | 重OECD库存跟踪，最及时；月报"STEO"覆盖全球 |
| **EIA**（美国能源信息署） | 美国（消费国） | 美国周报数据最精细；附"剩余产能"指标（须能在90天内持续供给9个月以上才算） |
| **OPEC**月报 | 产油国视角 | 核心指标"Call on OPEC"（全球需求减非OPEC供给减OPEC NGL）；是OPEC开会决策的技术依据 |

**跟踪方法**：不必自建模型，跟踪三大机构月报的边际变化即可。关键看同一机构前后月对同一年份预测的修订方向。

### 地缘分区：东西市场各自平衡

佘建跃以**苏伊士运河**为东西分界线，并通过BP年度能源统计进一步细分为6区（亚太、中东、非洲、欧洲、欧亚、中南美、北美）。结构性结论：

- **亚太**：产量极少（中国保底2亿吨/年），需求高速增长 → 严重供不应求
- **中东**：供给充裕，消费有限 → 天然净出口
- **北美**：2009年页岩油革命后从净进口转为净出口平衡；2015年后北美整体净出口
- **欧洲（+欧亚）**：供需大致平衡

**东区**（亚太+中东）：中东供给与亚太需求互补，区内大致平衡
**西区**（北美+欧洲）：页岩油革命后也达到各自平衡

→ 当东西两区各自平衡时，东西基准油价差（布伦特-迪拜）呈**区间震荡**；若某区单边失衡，价差将突破区间，套利窗口打开，贸易流向改变，重塑两区基本面，价格回归。

### 长期供需规律

- **需求**：石油年需求增量（万桶/日）≈ 全球GDP增速（%）÷ 2 × 100
  - 例：GDP增3%，石油需求增约150万桶/日
  - 长期还受人口结构驱动（人口增长地区对应需求增量）
  - 全球需求增量约70%来自中国、印度、美国三国

- **供给**：长期供给看上游投资（CAPEX）；2014年油价暴跌导致全球上游投资急剧收缩，2016年见底，此后复苏但未回到2014年高峰水平 → 供给侧中期存在投资不足的结构性问题

- **页岩油成本**：远期曲线的远端价格（如WTI 3-5年期合约，约51-53美元/桶）=美国页岩油行业平均成本；做市商以折现成本定价远端合约。这是读取页岩油边际成本最简单的方法。

### 三巨头时代取代OPEC主导地位

2019年三大产油国产量均超1000万桶/日：
- **美国**：1230万桶/日（原油+凝析油），加上NGL（450万桶/日）共1650万桶/日，基本实现能源独立
- **沙特**：约980-1100万桶/日（最高曾达1100）
- **俄罗斯**：约1100万桶/日

三者合计约占全球总产量1/3，且横跨东西两区（俄罗斯油同时供应欧洲和亚洲），使传统OPEC/非OPEC二元框架面临挑战。

## 价差结构

### 东西价差（布伦特 vs 迪拜）

**区间波动公式**（各自平衡条件下）：
```
-运费 + 品质差 < 布伦特-迪拜 < 运费 + 品质差
```
实证区间约1-4美元，中轴约2.5美元（即布伦特比迪拜贵约2.5美元），对应品质差；±1.5美元对应运费。

**区间突破信号**：
- 价差 > 4美元：中东原油相对便宜，欧洲炼厂加入东向采购队伍 → 东向套利窗口打开 → 价差收窄
- 价差 < 1美元：西非/北海原油相对便宜，亚洲炼厂加入西向采购 → 西向套利窗口打开 → 价差扩大

**EFS（期货转掉期）**是布伦特-迪拜价差的主要交易工具，也是判断跨区套利方向的实时指标。

**IMO2020船燃新规影响**：由于低硫柴油的裂解价差相对高硫燃料油偏强，布伦特（低硫轻质）与迪拜（中质含硫）的品质差有扩大趋势，未来东西价差中轴上移。

### 跨大西洋价差（WTI vs 布伦特）

2009年之前（美国高度依赖进口）：布伦特-WTI基本稳定在±1-2美元，类似区间震荡（实为"地板模型"，WTI有天然价格下限因为美国缺油不能让价格太低）。

2011-2014年：页岩油暴涨但库欣地区管道设施未跟上，形成内陆"堰塞"，WTI对布伦特贴水最大扩至-28美元。这是物流约束导致的结构性价差，而非基本面永久改变。

**每一次管道投产（Seaway管线反输、Seaway复线等）都伴随价差的快速回归。**

2014年后：页岩油主产区从巴肯南移至德州二叠纪盆地（离海边更近），物流条件改善，WTI-布伦特价差重新趋近区间震荡（约-5至0美元），前提是管道能力匹配。

**"德州新三角"**：二叠纪盆地-库欣-休斯顿三角形，当二叠纪到休斯顿直通管线不足时，油必须绕道库欣，运费叠加约6美元/桶（3+3美元两段管费）。

### 裂解价差的规律

**大宗过程工业财务平衡公式**：
```
产物价值 = 原料成本 + 费用 + 毛利
Σ (收率_i × 价格_i) = 原油价格 + 升贴水 + 费用 + 毛利
```
因为各产品收率之和≈100%，变形后：
```
Σ 收率_i × 裂解价差_i = 升贴水 + 费用 + 毛利
```

**核心规律**（数学推导的必然结果）：
- **主产品裂解价差与副产品裂解价差负相关**
  - 炼油业：轻油（汽柴油）= 主产品；重油（燃料油）= 副产品
  - 柴油裂解价差 ↑ → 燃料油裂解价差 ↓（负相关）
  - 小收率品种的裂解价差波动幅度 = 大收率品种的n倍（反比关系）

- **柴油裂解价差与油价关联性最强**；汽油裂解价差关联性弱
  - 原因：柴油离原油更近（直馏柴油→加氢），汽油是多组分调和品，离原油更远
  - 柴油多用于重型机械/工业，与经济周期联动更紧密；汽油反映生活水平和消费

- **主副产品定义标准**：使用功能能被原料替代的是副产品；不能替代的是主产品
  - 豆粕可用大豆替代（饲料）→ 副产品；豆油不能用大豆炒菜 → 主产品
  - 燃料油可用原油烧船 → 副产品；汽柴油不能直接用原油发动汽车 → 主产品

### 跨期价差（远期曲线结构）

**远期曲线的形状与意义**：

| 结构 | 含义 | 形成机制 |
|------|------|----------|
| **Contango**（近低远高） | 供大于求 | 近端供给压力大、交易员急于卖近月，远端基本面不确定、价格跌少 |
| **Backwardation**（近高远低） | 供不应求 | 近端抢购现货、远端无人争，近月涨多 |

**正反馈机制**：
- 深度Contango → 正套（买现货+抛远期）有利 → 库存增加 → 缓冲供给过剩 → Contango自动修复（但绝对价格仍偏低）
- 深度Backwardation → 反套（卖现货+买远期）有利 → 库存快速去化 → 加速供不应求 → 绝对价格上涨

**仓储视角**：
- Contango时：缸区/浮仓需求旺盛，租金高；收购库区要趁Backwardation（便宜）
- 炼厂库存策略：Contango时增库（过剩原油便宜）；Backwardation时减库（卖高买低的反套）

## 地缘政治与油价

### 石油的三个独特属性

1. **储量分布高度集中**：中东储量占全球50%以上，且这一格局亿万年地质决定，不会改变 → 中东永无宁日
2. **相对稀缺性**：石油储采比低于煤炭和天然气；相对最稀缺的化石能源
3. **地缘供需严重失衡**：煤炭/天然气各区供需大致均匀，石油的供应堆积图与需求堆积图完全无法重叠 → "为石油而战"是历史必然

### 石油历史的三个阶段

1. **七姐妹垄断时代**（1860s-1960）：标准石油拆分后的埃克森、美孚、BP、壳牌、雪佛龙、德士古、海湾石油七家公司垄断上游定价，产油国获益极少
2. **OPEC觉醒时代**（1960-1990s）：1960年OPEC成立，1973年第一次石油危机（赎罪日战争）→ 石油禁运 → 油价暴涨；北海油田1970年发现、1980年代大规模开发 → 供应宽松 → 现货市场兴起 → WTI期货1988年上市
3. **三巨头博弈时代**（2009-今）：页岩油革命重塑地缘格局；美国从"为石油而战"转为袖手旁观/代理人战争；沙特从产量换收入转向资产上市（沙特阿美）换收入

### 地缘政治的"乱而不断"潜规则

核心逻辑：**霍尔木兹海峡封锁是全球不可承受之痛**，包括伊朗自己也不能承受断供的经济代价。因此：
- 地缘紧张是常态，但真正断供极其罕见
- 各方最优选择：制造恐慌溢价（语言/无人机小规模攻击），而非真正断供
- 100万桶/日断供可用沙特剩余产能覆盖，油价约涨50%（至约100美元）；200万桶/日以上难覆盖

**美国在地缘中的角色转变**：
- 页岩油革命前：美国靠武力保护中东石油安全（海湾战争1991年、伊拉克战争2003年）
- 页岩油革命后：美国能源独立，但不允许世界太平——因为中东/亚太仍依赖石油进口，这是美国的战略杠杆
- 特朗普推特喊"油价太高"其实是掩护，美国需要高油价支撑页岩油开采成本（财政平衡油价）

### 沙特的战略转型

- 传统模式：产量换收入（卖原油）
- 新模式：储量换收入（资产上市——沙特阿美IPO，用油田储量做抵押融资）
- 含义：沙特有维持高油价的内在动机（融资估值越高）；再也不会用低油价打击页岩油
- 2019年起的策略：维护OPEC+框架，与俄罗斯协同减产，以供给管理维持价格区间

## 实货贸易与套利

### 实货原油基础知识

**计量单位**：桶（Barrel）= 42加仑 ≈ 159升；历史来源：美国第一口油井（宾州，1859年德雷克井）用木桶装油

**品质分级**：
- **API重度**：>10漂浮，<10沉入水底；数值越大越轻；好油API高（如WTI约39-42，布伦特约38，迪拜约31-33）
- **硫含量**：低硫（<0.5%，sweet）为好油；高硫（>0.5%，sour）为差油；脱硫需要额外成本
- 好油 = 高API + 低硫；差油 = 低API + 高硫（重质高硫原油）

**运输方式**：管道（最便宜、密闭安全）> 油轮（灵活）> 火车（辅助）。VLCC（超大型油轮）载重25万吨以上，一次可装180万桶。

### 升贴水（Differential）的决定因素

升贴水 = 品质差 + 时间价值

- **品质差**（稳定部分）：由各产品收率×裂解价差加权决定，随裂解价差结构变化
- **时间价值**（浮动部分）：由EFS（布伦特-迪拜价差）反映跨区供需平衡

实证案例：
- 当EFS（布伦特-迪拜）价差扩大，迪拜系原油显得便宜 → 俄罗斯ESPO升水上涨（迪拜计价，油质更轻）
- 当EFS价差收窄，西非原油显得便宜 → 亚洲炼厂转买西非 → 西非贴水（布伦特计价）走强

### 套利与交割的实务

**正套（Carry Trade）**：在Contango结构下
- 买现货 + 抛远期期货 = 锁定价差收益
- 前提：价差能覆盖仓储成本（租库/浮仓费）+ 资金成本
- 效果：大量库存吸纳过剩，避免油价无限下跌

**反套（Reverse Carry）**：在Backwardation结构下
- 卖现货（高价）+ 买远期期货（低价）= 锁定收益
- 同时将仓库出租（获仓储收益）+ 资金理财（获财务收益）
- 效果：库存快速释放，加剧近月供应紧张，价格进一步上涨（正反馈）

**贸易员视角**：贸易员核心工作不是判断油价绝对涨跌，而是判断**升贴水的走势**（什么时候买到的贴水可以转手卖出更高的贴水）。这基于对东西两区相对供需平衡关系的理解。

### IMO 2020船燃新规（2019年当时的热点事件）

国际海事组织（IMO）规定自2020年1月1日起，船舶燃料油硫含量不得超过0.5%（原标准3.5%）。

三种应对路径（各占约1/3比例）：
1. 安装尾气脱硫装置（洗涤器）继续烧高硫燃料油
2. 炼厂升级装置，减产高硫燃料油、增产低硫产品
3. 改烧低硫柴油/0.5%调合燃料油

**影响链条**：
- 高硫燃料油供应过剩 → 高硫燃料油裂解价差暴跌（远期曲线已提前反映）
- 主副产品负相关 → 柴油裂解价差走强
- 烧柴油的转化率约50%（1吨柴油替代1吨燃料油，但需要2吨原油生产1吨柴油） → 原油总需求增加
