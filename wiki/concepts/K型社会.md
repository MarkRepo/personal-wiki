## 定义

K型社会指社会经济复苏或发展呈现"K"字形分化的状态：上层富人资产持续增值（K的上升腿），下层普通人收入停滞甚至下降（K的下降腿）。这一概念被美联储前主席耶伦用于描述疫后经济复苏的分化特征，[[entities/付鹏]]在《[[entities/见证逆潮]]》中将其上升为全球化和金融化的必然产物。

## 核心原理

### 形成机制

K型社会的形成经历三个阶段（按金融化程度递进）：

**第一阶段 —— 劳动价值阶段**：
- 体力劳动者间收入差异不大，收入差异来自技能和职业分工
- 不会形成显著贫富差距，社会趋向"橄榄形"

**第二阶段 —— 债务价值阶段**：
- 企业利用负债获取资产超额投资回报
- 掌握融资能力的人开始拉大收入差距

**第三阶段 —— 杠杆价值阶段**：
- 杠杆运用迅速提高产出和收入斜率
- 掌握负债和杠杆者掌握财富分配权
- 社会从"橄榄形"分化为"K型"

**关键洞察**：债务和杠杆应用的先后顺序决定社会分层——先掌握者获得财富，后掌握者承担债务负担。

### 传导机制

```
全球化产业转移 → 底层劳动失业/薪资下降
        +
金融自由化 → 资产价格通胀 → 财富向金融资产持有者集中
        +
低利率政策 → 富者更容易融资购买资产 → 马太效应
        ↓
K型分化形成
```

### "利率病"与K型的自我强化

```
降息 → 资产升值 → 富人更富 → 贫富分化加剧 → 需求更弱 → 进一步降息
→ 央行被高资产价格绑架 → 无法加息（怕资产崩溃）→ 利率永远无法正常化
```

这是付鹏所说的"利率病"——央行陷入无法退出的困境。

### 货币政策的结构性失效

- 宽松政策首先流向富人和高利润企业 → 资产价格上升
- 普通民众收入停滞 → 消费倾向虽高但能力不足
- 金融资产由富人持有 → 资产升值 → 财富进一步集中
- 储蓄 > 投资 → 利率越来越低 → 恶性循环

## 应用场景

- **理解货币政策失效**：为什么量化宽松无法刺激实体经济增长
- **分析社会矛盾**：民粹主义、逆全球化的经济根源
- **资产配置**：K型结构下金融资产与实体经济的分离
- **政策评估**：评判财政政策和货币政策的分配效应

## 局限性

- K型分化在不同国家表现差异大，不宜过度简化
- 未充分考虑技术创新可能带来的阶层流动
- 库兹涅茨曲线是否真正失效仍有争议——部分学者认为只是周期拉长
- 对中等收入群体的分析不够细致，"K"字形可能是过度简化
