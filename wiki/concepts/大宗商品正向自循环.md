## 定义
大宗商品正向自循环是指在外部变量（宏观政策、供给冲击等）扭转市场悲观预期后，远期曲线从深度Contango状态逐步修复，通过市场参与者行为的相互强化，最终推动商品价格持续上涨、曲线转为Backwardation的正反馈过程。是付鹏大宗商品周期分析框架中的核心逻辑。

## 核心驱动因素
正向自循环的启动需要两个关键前提：
1. **外部变量触发预期扭转**：
   - 宏观因素：政策宽松、经济刺激计划等激发需求预期
   - 微观因素：供给收缩、自然灾害、停产检修等改变供给预期
2. **基差先行验证**：现货价格不再跟随期货下跌，期现基差大幅拉大，反映真实供需已经触底

## 循环传导路径
```
外部变量触发 → 预期扭转 → 基差修复 → 曲线从Super Contango回归Contango →
套利者介入（买近抛远） → 贸易商建立虚拟库存 → 近月价格上涨 →
预期进一步强化 → 贸易商开始囤货建立真实库存 → 生产商惜售 →
供应紧张预期升温 → 曲线转为Backwardation → 价格加速上涨 →
形成Super Backwardation → 正向循环达到顶峰
```

## 各阶段参与者行为
1. **初期修复阶段**：高级交易者和月差套利者率先介入，通过买近抛远获取基差修复收益，贸易商开始在期货上建立虚拟库存
2. **预期强化阶段**：现货商和贸易商开始囤货，锁定库存导致真实供应进一步紧张，生产商开始惜售并调高出厂价
3. **加速上涨阶段**：投机资金大量涌入，近月合约成交量和持仓量大幅放大，曲线陡峭Backwardation
4. **顶峰反转阶段**：需求无法跟进，期现基差转负，贸易商开始抛近月交割，生产商在远月大量套保，循环反转进入负向周期

## 风险点
1. **逼仓风险**：Super Backwardation阶段容易出现多逼空行情，小品种商品逼仓风险更高
2. **预期证伪风险**：如果外部变量带来的预期最终没有被基本面数据验证，循环会提前中断
3. **政策干预风险**：行政调控价格会直接打破自循环逻辑

## 应用场景
- 周期底部识别：基差先行修复是大宗商品价格触底的领先信号
- 交易策略制定：正向循环初期适合单边做多近月合约，中期适合月差套利策略
- 顶部信号识别：Super Backwardation叠加现货需求疲软是周期见顶信号
