## 定义
大宗商品正向自循环是指在外部变量（宏观政策、供给冲击等）扭转市场悲观预期后，远期曲线从深度Contango状态逐步修复，通过市场参与者行为的相互强化，最终推动商品价格持续上涨、曲线转为Backwardation的正反馈过程。是付鹏大宗商品周期分析框架中的核心逻辑。

## 核心驱动因素
正向自循环的启动需要两个关键前提：
1. **外部变量触发预期扭转**：
   - 宏观因素：政策宽松、经济刺激计划等激发需求预期
   - 微观因素：供给收缩、自然灾害、停产检修等改变供给预期
2. **基差先行验证**：现货价格不再跟随期货下跌，期现基差大幅拉大，反映真实供需已经触底

## 循环传导路径
```
外部变量触发 → 预期扭转 → 基差修复 → 曲线从Super Contango回归Contango →
套利者介入（买近抛远） → 贸易商建立虚拟库存 → 近月价格上涨 →
预期进一步强化 → 贸易商开始囤货建立真实库存 → 生产商惜售 →
供应紧张预期升温 → 曲线转为Backwardation → 价格加速上涨 →
形成Super Backwardation → 正向循环达到顶峰
```

## 各阶段参与者行为
1. **初期修复阶段**：高级交易者和月差套利者率先介入，通过买近抛远获取基差修复收益，贸易商开始在期货上建立虚拟库存
2. **预期强化阶段**：现货商和贸易商开始囤货，锁定库存导致真实供应进一步紧张，生产商开始惜售并调高出厂价
3. **加速上涨阶段**：投机资金大量涌入，近月合约成交量和持仓量大幅放大，曲线陡峭Backwardation
4. **顶峰反转阶段**：需求无法跟进，期现基差转负，贸易商开始抛近月交割，生产商在远月大量套保，循环反转进入负向周期

## 风险点
1. **逼仓风险（Corner Risk）**：Super Backwardation阶段容易出现多逼空行情，小品种商品逼仓风险更高（详见下方"Corner Risk完整分析"章节）
2. **预期证伪风险**：如果外部变量带来的预期最终没有被基本面数据验证，循环会提前中断
3. **政策干预风险**：行政调控价格会直接打破自循环逻辑

## Corner Risk（逼仓风险）完整分析

逼仓（Corner Risk）是正向自循环接近Super Backwardation时的高风险阶段，需要从以下5个维度综合评估（来源：付鹏，2016-03-27）：

1. **社会库存**：期货注册仓单以外的流通总量，包括贸易商持有的显性和隐性库存
2. **隐性库存**：贸易商和中间环节持有的未注册库存规模；隐性库存越少，逼仓成功概率越高
3. **期现基差（basis）**：期货近月价格与现货价格的差值；basis为负（negative basis）时，是逼仓从多逼空向空逼多转换的信号
4. **交割库容**：交割仓库能否容纳足够多的货物；交割库容不足会加剧多头挤压压力
5. **交割月持仓**：临近交割月时的未平仓持仓量；集中持仓越大，逼仓风险越高

**农产品 vs 工业品的逼仓难度差异**（来源：付鹏，2016-03-27）：
- **农产品squeeze更容易**：季节性供给集中、不易长期保存、现货流通渠道分散，空头难以大规模调货交割
- **工业品原则上不容易**：金属等工业品相对容易获取、保存方便，全球产能充足；**例外情况**：某些小众工业品中有大型持仓者控制着现货核心流通环节，此时逼仓风险大幅提高
- **关键原则**：现货盘子越大，Corner Risk越低；在高流动性泛滥的环境下，全球真正不容易被逼仓的只剩下少数几个商品巨无霸，中国三大交易所交易的大部分商品都存在Corner Risk

**多逼空 → 空逼多的转换机制**（来源：付鹏，2016-03-27）：
1. 正向自循环顶部阶段，囤货观望的贸易商、投机多头集中持有大量近月多单
2. 一旦现货在实际需求无法跟进时，期现基差转为negative（现货无法跟上近月期货步伐）
3. 此时：囤货的贸易商将商品率先抛到近月合约上 → 多逼空反转为空逼多（吸引原来集中在现货和远端的抛盘向近月集中）
4. 贸易商赚的已经赚到了：抛近月交割，远月买回建立虚拟库存，negative spread也一并吃进；生产商同步快速将库存或近月产量保值到近月，甚至直接入仓交割
5. **防范方法**：手上没有商品时，千万不要轻易在Super Backwardation阶段做空近月合约——近月合约到期日是时间约束，没有实货交割能力的空头极易被Corner

**顶部反转的精确触发**（来源：付鹏，2016-03-27）：
- Negative basis出现后，贸易商的最优策略是：**抛出近月合约交割现货 + 远月买回建立虚拟库存**（负价差收益也一并落袋）
- 生产商：前期调高出厂价、封盘惜售；一旦需求无法跟进且远月利润仍存在，生产商开始在远月大量套保，甚至逐步将远期产出保值到远月合约上
- 循环逆转的触发点：**现货价格无法追上期货近月合约的步伐**（negative basis），此即正向循环的真正顶部

## 螺纹钢案例（2015-2016年正向自循环应用）

以下是付鹏对2015年6月至2016年2月螺纹钢期现价格走势的详细分析，是正向/负向自循环理论的完整实操案例（来源：付鹏，2016-03-29）：

| 时段 | 关键特征 | 曲线变化 |
|------|----------|----------|
| 2015.6.12-7.8 | 股灾叠加需求悲观，铁矿石三连跌停传导螺纹，现货深度贴水期货 | Contango整体平滑下移 |
| 2015.7.8-8.10 | 中信救市托底，负向循环被政策信心短暂阻断；刚性需求补库存推现货快速走高 | 近月端略微平滑，远端仍反映悲观 |
| 2015.8.10-8.19 | 弱反弹夭折：现货超跌反弹后无法引发链条跟随，第一次正向转折尝试失败 | 现货向期货回归 |
| 2015.8.19-9.30 | 811人民币贬值近4%叠加A股再次暴跌，过度悲观预期全面蔓延，全球系统性风险升高 | Contango再次整体下移 |
| 2015.9.30-11.11 | 恐慌缓解（美联储推迟加息、A股反弹、人民币企稳），现货略升，期货仍谨慎，曲线悄悄出现轻微back | 远期曲线从Contango转为轻微Back |
| 2015.11.11-12.7 | 中国三季度经济无亮点 + 人民币离岸压力 → 市场信心再次瓦解，曲线从轻微Back被打回Contango | 现货领跌 |
| 2015.12.7-2016.1.15 | 美联储加息落地，但Contango曲线未再下跌，现货反而反弹100块——**绝对价格1600-1700是黑色产业链出清的相对平衡点** | Contango曲线稳定，基本面刚性开始主导 |
| 2016.1.15-2.5 | 中美对话（汪洋与美国财长通话）+ PBOC收拾离岸空头 + 政府增加杠杆稳需求，**等待已久的外部变量终于出现** | Contango进一步平滑，等待数据确认 |
| 2016.2.5-2.22 | 1月新增信贷数据 + 一线城市房价疯狂上涨 + 新开工大幅增加，**外部变量触发正向自循环启动** | 曲线从Contango flat快速转为Backwardation，期货带动现货上涨 |

**案例核心教训**：
1. 同一绝对价格水平可以多次被"过度悲观预期"压过真正的基本面刚性，底部出清需要时间
2. 外部变量（政策信号、宏观数据）是正向循环启动的关键触发点，没有外部变量，基本面刚性只是远端支撑
3. 现货价格的坚挺（negative basis消失或basis转正）是底部信号的领先指标

## 应用场景
- 周期底部识别：基差先行修复是大宗商品价格触底的领先信号
- 交易策略制定：正向循环初期适合单边做多近月合约，中期适合月差套利策略
- 顶部信号识别：Super Backwardation叠加现货需求疲软是周期见顶信号
