## 定义

问题归属（Problem Ownership），P.E.T.中判断"谁拥有问题"的原则。当孩子的某个行为需要回应时，先判断这个行为影响了谁的需求——谁的需求受到影响，谁就拥有问题，然后用对应的技巧回应。这是整个P.E.T.方法论的路由枢纽。

## 核心原理

### 三种问题归属情况

**1. 孩子拥有问题**
- 孩子的需求受阻，父母需求未受影响
- 孩子表达情绪（沮丧、痛苦、失望、害怕）
- → 使用 **[[concepts/积极倾听]]**
- 例：孩子被朋友冷落、没进球队、害怕打针、两门功课不及格

**2. 父母拥有问题**
- 孩子的行为对父母的需求产生切实具体的影响
- 父母感到担心、疲惫、沮丧、受干扰
- → 使用 **[[concepts/我-信息]]**
- 例：孩子把刚收拾好的厨房弄乱、放学晚归没打电话

**3. 双方都有问题（关系拥有问题）**
- 双方需求都受影响，积极倾听和我-信息都无法解决
- → 使用 **[[concepts/没有输家方法]]（第三法）**
- 例：父母需要安静工作，孩子需要在家玩耍

**无问题区**：孩子的行为可接纳，双方均无需求受阻，是亲子关系中快乐的共处时光，无需任何干预。

### 判断流程

```
孩子的行为在接纳线上方？
  → 是：无问题区，无需干预
  → 否：谁的需求受影响？
      → 只有孩子的需求受阻 → 孩子拥有问题 → 积极倾听
      → 只有父母需求受阻 → 父母拥有问题 → 我-信息
      → 双方需求都受阻 → 关系问题 → 没有输家方法
```

### P.E.T.的总体目标

通过掌握三种技巧，缩小两端的问题区，扩大中间的"无问题区"。父母学员常说："我能分辨谁处在问题区，这是最有价值的事。我的孩子拥有问题，而我不需要为此负责——这么多年来我一直将它们揽在自己肩上。"

## 常见误用

**孩子有问题时强行干预**：孩子面临困境，父母急于给建议和解决方案——表面上是在帮忙，实际上剥夺了孩子自己解决问题的机会，且传递了"你没能力解决自己的问题"的信息。

**父母有问题时却用倾听**：父母的需求受影响，却只是"忍着"或"积极倾听"孩子的辩解，而不发出"我-信息"。劳埃德太太对女儿布莱娜所有要求妥协，长期压抑自己权利，最终对女儿产生强烈怨恨，而布莱娜变得极度自私——孩子的自私是父母长期让渡问题归属的直接后果。

## 局限性

- 行为窗口的接纳线是动态的，同一行为的问题归属也可能随情境变化
- 需要父母有足够的自我觉察来区分"我真的被干扰了"和"我只是觉得不应该"
- 问题归属可以中途转换：父母发"我-信息"后，孩子产生情绪反应，需要临时切换为积极倾听（换挡技术）
