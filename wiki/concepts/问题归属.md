## 定义
问题归属（Problem Ownership），P.E.T.中判断"谁拥有问题"的原则。根据行为窗口模型，当孩子的行为越过接纳线时，决定采用哪种沟通技巧的关键在于：谁的需求受到了影响，谁就拥有问题。

## 核心原理

### 三种问题归属情况

**1. 孩子拥有问题**
- 孩子的需求受阻，父母需求未受影响
- 行为在接纳线下方，但不干扰父母
- → 使用**积极倾听**
- 例：孩子被朋友冷落而难过

**2. 父母拥有问题**
- 孩子的行为干扰了父母满足自身需求
- 行为越过接纳线进入不可接纳区，且对父母造成实际影响
- → 使用**我-信息**
- 例：孩子把刚收拾好的厨房弄乱

**3. 双方都有问题（关系拥有问题）**
- 积极倾听和我-信息都无法解决
- 父母和孩子都有需求冲突
- → 使用**没有输家方法（第三法）**
- 例：父母需要安静工作，孩子需要在家玩耍

### 判断流程图
```
孩子的行为在接纳线上方？
  → 是：无需干预
  → 否：谁的受影响？
      → 孩子的需求受阻 → 孩子拥有问题 → 积极倾听
      → 父母需求受阻 → 父母拥有问题 → 我-信息
      → 双方需求都受阻 → 关系问题 → 第三法
```

## 应用场景
- 任何亲子互动前的快速判断工具
- 帮助父母避免"孩子没有问题时强行干预"
- 避免"父母有问题时却用倾听而非面质"

## 局限性
- 行为窗口的接纳线是动态的，问题归属也可能随时变化
- 需要父母有足够的自我觉察来区分"我真的被干扰了"和"我只是觉得不应该"
