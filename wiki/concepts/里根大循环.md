## 定义

里根大循环（Reagan Grand Cycle）指1980年代美国里根政府时期形成的"双赤字"经济循环模式：政府预算赤字与经常项目赤字相互强化，通过高利率吸引国际资本流入，形成"强经济+强货币+大赤字+巨大贸易逆差"的自我强化循环。[[entities/付鹏]]在《[[entities/见证逆潮]]》中将其视为理解美元霸权和全球化金融秩序的关键。

## 核心原理

### 正反馈路径

```
财政赤字扩张 → 政府支出增加 → 需求扩张 → 经济增长
→ 利率上升 → 国际资本流入 → 美元升值
→ 资本继续流入 → 可承受更大赤字 → 继续扩张
```

### 历史背景：美国经济转型（1980-1995）

里根大循环建立在四个关键政策基础之上：

1. **去泡沫化**：沃尔克（Volcker）以极高利率（联邦基金利率一度超20%）控制通胀，代价是失业和衰退
2. **减税放松**：1981年减税法案将最高税率从70%降至50%，放松石油、电信、金融部门管制
3. **产业升级**：高科技/信息产业取代低端制造业，后者向海外迁移
4. **美元霸权重塑**：从金本位转向石油美元体系，通过多元信用体系（科技、金融、军事）维持美元地位

### 石油美元体系

```
现代化 → 工业化 → 石油需求 → 美元计价 → 美元需求
→ 美国掌握全球"负债权力" → 可持续发行美元债务
```

三个发展阶段：
- 1974-1985年：建立期
- 1986-1999年：克林顿时期调整
- 2000-2013年：衰落期

### 与全球化的关系

里根大循环为[[concepts/三级分工体系]]提供了货币基础：
- 美国作为消费国通过美元霸权维持经常项目逆差
- 生产国（中国等）积累美元外汇储备，回购美债
- 形成闭环：美国消费 → 生产国出口 → 美元回流美国

## 应用场景

- **理解美元周期**：美元强弱与全球资本流动的关系
- **分析新兴市场危机**：美国加息/去杠杆时新兴市场面临的资本外流压力
- **解读中美关系**：贸易逆差和债务关系的结构性根源
- **FICC资产定价**：利率曲线Contango结构的形成机制

## 局限性

- 双赤字模式的可持续性依赖美元霸权地位，若美元地位动摇则循环断裂
- 高利率吸引资本的同时可能刺破国内杠杆，引发金融危机
- 理论较为美国中心主义，未充分考虑欧元、人民币等替代货币的影响
- 里根时期的政策组合在当前财政约束下可能难以复制
