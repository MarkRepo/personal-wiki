## 定义

付鹏对全球化内在矛盾的系统分析：全球化通过贸易自由化创造了巨大财富，但同时制造了难以持续的**结构性失衡**——制造国（中国、德国）储蓄过度，消费国（美国）债务过度，资本在两者之间循环流动，最终因这种失衡难以为继而进入重塑阶段。

## 核心原理

### 全球化的两类国家

**A型（过度储蓄国）**：
- 代表：中国、德国
- 特征：大量固定资产投资，居民储蓄高，经常账户盈余
- 债务结构：主要在企业和政府（基础设施、制造业扩张）
- 产业：原材料进口 + 制成品出口

**B型（过度消费国）**：
- 代表：美国
- 特征：债务扩张驱动消费，居民储蓄低，经常账户赤字
- 债务结构：主要在居民（房贷、信用卡）
- 产业：服务业为主，制造业空心化

**全球失衡图示（付鹏课程核心图）**：
```
                债务扩张（制造需求）
           ┌──────────────────────────┐
           │         欧美              │
           │    过度消费               │
           │   居民|金融|企业           │
           │      债务↑               │
           └──────────────────────────┘
                  ↑贸易逆差           ↓美元回流
           ┌──────────────────────────┐
           │         中国              │
           │    过度储蓄               │
           │   居民|企业|政府           │
           │      储蓄↑               │
           └──────────────────────────┘
                ↑原材料  ↑原油
                进口国
```

### "中国冲击"（China Shock）的失衡效应

全球化放大了失衡的具体机制：
1. 中国廉价劳动力大量进入全球供应链
2. 全球制成品价格中枢被压低
3. 美国等发达国家制造业工人的工资实际上没有增长（甚至下降）
4. 但美国GDP仍增长——因为服务业和金融业扩张
5. 结果：底层居民（制造业失业工人）收入停滞，高层（金融、科技）收入大涨
6. 这就是[[concepts/K型社会]]的形成机制

> "这20年全球化发展以后，在中间的分配环节上出了非常大的问题，大部分底层人民的收入其实是没有增长的。"（付鹏，2019年大师课）

### 全球化失衡的四大体现

1. **贸易失衡**：中国对美国巨额贸易顺差持续20年以上
2. **资本倒流**：中国出口获得的美元，通过购买美国国债流回美国，压低了美国长端利率
3. **货币政策扭曲**：低利率环境促使美国居民过度借贷（次贷危机根源之一）
4. **分配失衡**：全球化红利主要流向资本持有者和跨国企业，底层劳动者受损

### 2020年后的全球重塑方向

付鹏在2019年底预判的重塑路径（事后证明基本准确）：

**A侧（过度储蓄国的压力）**：
- 产业转移：海外企业减少在华投资减少
- 先前大量投资导致的企业债务压力
- 中国企业投资产能过剩
- 房地产和居民负债增加，储蓄减少
- 顺差减少，本币贬值压力

**B侧（过度消费国的调整）**：
- 疫情加剧反思：缺少投资和生产
- 进一步推动产业转移
- 政治上主义更加的强烈
- 对于全球贸易壁垒增加，贸易的阻断
- 未来要逆转贸易逆差（增加生产，减少进口）
- 海外美元储蓄减少，海外美元供给需求增加
- 贸易差和产业的资产矛盾，需要产业重新分工
- 财政赤字需要大幅提高，军事支出减少
- 政府主导的固定资产投资加大
- 吸引海外投资者和海外移民

### 特朗普政策的宏观解读

付鹏将特朗普政策定性为"全球经济一体化的终结者"：

> "特朗普的政策：你的通胀是你的问题，我的通胀是我的问题。"

这句话的实质：
- 美国不再愿意为全球提供需求（不再充当消费者）
- 美国不再愿意让美元充当全球流动性
- 美国要求贸易伙伴"各负其责"，不再维护全球化框架

**历史平行**：
- 1970年代，美国面临类似挑战（日本制造崛起，美国制造业空心化）
- 当时选择：关闭黄金兑换窗口（1971年布雷顿森林崩溃）+ 打贸易战
- 结果：经过15年调整（1970-1985），美国完成工业结构转型
- 现在同样：贸易战是15年调整期的开始，科技革命是最终的出路

## 应用场景

**预判产业转移方向**：
- 从中国向哪里转？（东南亚：劳动力密集型；墨西哥：近岸外包；美国：高端制造回流）
- 哪些行业首先受影响？（芯片、稀土、新能源等战略产业）

**配置大类资产**：
- 全球化失衡修复过程中，通胀压力可能上升（供应链效率下降，成本上升）
- 国债（安全资产）在动荡期受益
- 依赖全球化低成本的企业盈利承压

**理解中国资产**：
- A型国家（储蓄过多）在全球化退潮时面临：外需收缩、国内消费不足、债务压力
- 中国转型路径：从出口驱动→内需驱动，但这个转型需要收入分配改革（难题）

## 局限性

- 全球化失衡并非线性崩溃，可能以博弈均衡的形式维持相当长时间
- "新殖民地生产"模式存在多方利益，打破成本高
- 产业转移的速度和规模受制于基础设施、供应链配套的实际约束
- 2020年疫情、2022年俄乌战争等黑天鹅事件加速了部分变化，超出2019年的预判框架
