## 定义

国内大循环（Domestic Circulation / Dual Circulation）指2020年中央提出的"以国内大循环为主体、国内国际双循环相互促进的新发展格局"，针对长期以来"重投资重出口轻消费"导致的内外失衡问题。[[entities/兰小欢]]在《[[synthesis/置身事内读书笔记]]》中将其置于国内消费不足、产能过剩、中美贸易冲突的大背景下系统分析。

## 核心原理

### 问题诊断：为什么消费不足？

居民消费占GDP比重长期偏低（2018年约44%），远低于发达国家（60-70%）。原因是双重的：

**1. 居民收入份额低**
- 可支配收入占GDP从约70%降至约60%
- [[concepts/分税制]]、招商引资模式、[[concepts/土地财政]]共同造成资源向企业和政府集中
- 劳动收入份额（相对于资本收入）持续下降

**2. 储蓄率高**
| 原因 | 机制 |
|------|------|
| 房价高企 | 首付积累需求抑制当期消费 |
| 社会保障不足 | 医疗、养老、教育自付比例高 → 预防性储蓄 |
| 计划生育 | 少子化 → 老龄化预期 → 储蓄防老 |
| 户籍制度 | 2亿城市移民无法享受当地公共服务 → 消费受限 |

### 失衡的传导链

```
国内消费不足 → 产能无法被国内吸收
→ 通过出口输出过剩产能 → 贸易顺差
→ 贸易伙伴国（美国）反弹 → 中美贸易冲突
→ 倒逼"双循环"转型
```

### 改革方向

1. **提高居民收入份额**：调整初次分配中劳动与资本的比例
2. **完善社会保障**：降低预防性储蓄动机
3. **要素市场改革**：户籍改革（释放农民工消费潜力）、土地流转（增加农民财产性收入）
4. **财政转型**：从"生产型"向"服务型"转变，增加民生支出

### 与"外循环"的关系

- 不是"闭关锁国"，而是以内需为主体、内外互促
- 出口仍然重要，但不再是增长的主引擎
- 进口同样重要——扩大进口有助于平衡国际关系

## 应用场景

- **理解当前中国经济政策方向**：为什么强调"扩大内需""共同富裕"
- **分析消费市场**：内循环意味着国内消费市场的战略性提升
- **评估贸易关系**：双循环框架下中美贸易摩擦的长期走向
- **投资含义**：消费升级、服务业、社会保障相关领域的长期机会

## 局限性

- "双循环"从提出到落地需要大量制度改革，进展可能缓慢
- 提高居民收入份额意味着压缩企业和政府份额，面临利益阻力
- 户籍改革涉及地方财政负担分配，推进困难
- 在全球经济减速背景下，扩大内需的难度增大
- 概念本身较为宏观，具体政策工具仍在探索中
