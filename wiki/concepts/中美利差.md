## 定义
中美利差是指中国和美国同期限国债收益率之间的差值，通常以10年期国债收益率作为基准，计算公式为：
> 中美利差 = 中国10年期国债收益率 - 美国10年期国债收益率

是分析跨境资本流动、人民币汇率走势、中美货币政策取向的核心指标，也是付鹏宏观分析框架中最重要的领先指标之一。

## 核心影响逻辑
中美利差的变化通过以下渠道影响宏观经济和金融市场：
1. **跨境资本流动**：利差是国际资本逐利的核心驱动因素，中美利差扩大时，国际资本更倾向于流入中国，获取更高的无风险收益；利差收窄甚至倒挂时，资本流出压力上升
2. **人民币汇率**：中美利差是人民币汇率的核心决定因素之一，利差扩大时人民币汇率趋于升值，利差收窄时人民币汇率趋于贬值
3. **货币政策独立性**：中美利差的水平直接限制了中国央行货币政策的操作空间，当利差过窄甚至倒挂时，央行降息会面临更大的资本流出和汇率贬值压力
4. **资产价格**：利差变化通过影响流动性和外资流向，直接影响A股、债券市场、房地产等大类资产的表现

## 付鹏分析框架中的核心规律
1. **舒适区间**：0.8%-1.5%是中美利差的舒适区间，在此区间内跨境资本流动基本平衡，汇率保持稳定，货币政策有足够的操作空间
2. **警戒线**：0.3%是中美利差的警戒线，低于这一水平时资本流出压力明显上升，汇率贬值压力加大
3. **倒挂阶段**：中美利差倒挂通常出现在美联储进入加息周期、中国央行进入宽松周期的阶段，对应人民币贬值压力最大的时期
4. **领先性**：中美利差的变化领先人民币汇率走势约1-3个月，领先跨境资本流动变化约1个月

## 应用场景
1. **汇率预判**：中美利差持续收窄时，应提前做好人民币贬值对冲；利差持续扩大时，人民币升值预期升温
2. **货币政策预判**：中美利差处于高位时，央行宽松政策空间充足；利差处于低位时，央行宽松政策会更加谨慎
3. **资产配置**：
   - 利差扩大阶段：外资流入增加，利好A股核心资产、利率债
   - 利差收窄阶段：外资流出压力上升，利好出口导向型行业、贵金属等避险资产
4. **跨境套利**：利差足够大时，国际资本可以通过借入低息美元、买入高息人民币资产进行无风险套利

## LC融资套利机制（2010-2013年实操细节）

2010-2013年中美利差持续扩大、人民币单边升值期间，大量套利资本通过以下完整流程进行跨境套利：

1. **境外设立壳公司**：同一主体在境外设立两家离岸公司（A公司、B公司），境内设立一家贸易公司（C公司）
2. **信用证开立**：A公司向C公司出口商品（如铜等大宗商品），境外银行开立以A公司为受益人的美元信用证（LC），期限90-180天
3. **离岸贴现**：A公司将信用证在离岸市场贴现获取美元，贴现成本约为低利率的美元短端利率（libor + 极小溢价）
4. **资金回流**：美元通过B公司或其他通道换成人民币汇回国内
5. **境内套利**：人民币资金在境内投入高收益理财产品，赚取人民币理财收益（年化6-8%）
6. **三重收益叠加**：① 息差（人民币理财利率 - 美元融资成本，约4-6%）；② 汇差（人民币单边升值，年化约2-3%）；③ 金属内外价差（境内商品价格高于境外价格的价差收益）
7. **循环往复**：信用证到期后清偿，商品原地未动（或在保税区转圈），资金完成一次套利后再次循环

渣打银行等外资行从中收取中间服务费，嘉能可等大宗贸易商提供现货和融资中介服务。付鹏估计这一时期的莱宝（Rauthenbach）等大宗贸易商，大量利润直接来自于此类套利交易。（来源：付鹏，2016-03-01）

## 大宗商品企业囚徒困境

矿业企业在大宗商品价格下跌期间，面临一个无法通过理性协调解决的非线性困境：

- **逻辑核心**：对企业而言，价格下跌时**重要的不是保价格，而是保现金流**
- **困境机制**：
  - 下游卖货卖不动，收入下降
  - 上游固定资产投资已经进去，无法立即回收
  - 银行看到股价下跌后，收紧信贷评级，进一步压缩融资能力
  - 被迫抛售库存以维持现金流，进一步压低市场价格
  - 价格下跌伤害资产负债表，进而伤害现金流量表——形成双杀
- **停产的真实触发条件**：不是绝对价格跌破成本线（例如铜的生产成本4000美元/吨），而是银行对企业的现金流施压、股价下跌导致融资能力下降，这时候才会停产；绝对价格跌到3500-3800美元也仍然会继续生产
- **同款逻辑**：与2008年金融危机雷曼倒闭的逻辑一模一样，诱因不同但机制相同
- **供应链连锁效应**：给矿业公司提供炸药、工程机械的所有企业股价全线见顶，如卡特彼勒、澳大利亚某炸药公司，是空头链式狙击的逻辑来源

（来源：付鹏，2016-03-01）

## 大宗商品周期循环表

终端消费到原材料的逐级传导机制（来源：付鹏，2016-03-01）：

```
终端消费需求扩张（如出口拉动）
  → 终端利润扩大 → 大量资本跟进终端
  → 原材料需求加大 → 资本向工业品转移
  → 工业品利润扩大 → 资本继续向上游原材料转移
  → 上游利润持续上移（因投产周期较长，供给滞后于需求）
  → 信贷扩张持续加大，全产业链超出真实需求的膨胀

→ 收缩开始：底端需求消失 → 中间环节供给过剩
  → 上游投资仍在扩张，需求已大幅下降（最惨烈阶段）
  → 产业链逐层死掉：底层→中游→上游

→ 底部信号：**上游巨头（BHP、Vale等）CDS走阔、不再做资本支出**
  → 这是商品价格真正构建底部的时间点
  → 新一轮周期：供给没跟上，讲供给故事而非需求故事
```

上游巨头 CDS（信用违约掉期）价格是判断大宗商品底部的领先信号；供给侧巨头重组并购、停止资本支出，是新一轮周期的起点。

## 资源国传导链

依附中国商品需求的资源国，在虚假繁荣退潮时按以下路径连锁崩溃（来源：付鹏，2016-03-01）：

1. **正循环阶段**：澳大利亚、加拿大、新西兰、蒙古、南非、哈萨克斯坦等资源国，借入低利率美元进行矿业投资扩张；汇率随本国资源出口繁荣升值，看似良好
2. **时滞效应**：矿业产出滞后于投资决策约3-5年，需求退去时供给仍在增加；投资需求先消失，真实消费需求后消失
3. **汇率失守**：套利资本流出，首先冲击市场化程度更高的经济体（巴西、南非等），然后向东南亚传导，最后压力到达中国
4. **产业链崩溃**：淡水河谷（Vale）、卡特彼勒全线空头；澳大利亚矿业炸药公司、工程机械公司等所有供应链企业股价全线见顶
5. **第二波传导**：第一波是出口和矿业支柱倒下，第二波是资产价格（房地产、股票）崩盘，与香港的路径完全相同
6. **贸易型企业的致命弱点**：嘉能可类贸易巨头，将高流动性现金流变成低流动性重资产（上游矿产），利差汇差消失后库存变成手上炸弹，面临流动性枯竭

## 2021年视角：传导路径的精确化与人民币领先性

（来源：付鹏，2021-12-12）

### 核心修正：利差是结果，汇率领先利差约 300 天

付鹏在 2021 年明确指出，**人民币汇率的变动大概领先中美利差 300 天左右**，而非利差领先汇率。传导路径先后依次为：
> 经常项目出口变化 → 内外经济状况变化（中债/美债分别反映内外需） → 中美利差变化

核心逻辑：中债代表中国内部需求，美债代表中国外部需求（即美国经济对中国出口的拉动强度），中美利差的走阔/收窄反映了美国经济相对中国的强弱，但这一反映滞后于汇率走势。这与付鹏框架中「分析顺序」一节的说明一致，但提供了更精确的领先时间量化（300天 vs 此前记录的1-3个月）。

### 六种情形组合框架
中美利差的分析需将「中债走向」与「美债走向」组合看，而非孤立看利差数值。可简化为六种情形（中债升/平/降 × 美债升/降的叉积），判断当下处于哪种组合，才能正确解读利差信号的含义和资产配置方向。

### 811汇改后的结构性再确认
- 2016年以来人民币汇率回归贸易/出口锚；
- 对出口型生产加工国家：PPIRM（原材料购进价格指数）对出口利润的蚕食远大于汇率波动，除非一次性贬值幅度极大，否则汇率贬值对出口的拉动效果非常有限；
- 汇率是外部需求和贸易政策变动的结果，不是原因。

### 2021年底前瞻判断（历史记录）
- 美债：远期需求预期放缓令长端上行空间有限，缩表加息预期令短端上行更快，美债收益率趋于扁平化；
- 中国：外部需求放缓若转内部刺激逆周期，中美利差将先收窄后走阔；
- 人民币升值空间收窄，将领先于利差变化率先小幅贬值；
- 全球不稳定性：疫情淡化后西方社会可能重回贸易保护主义路径。

## 历史案例
1. **2018年**：美联储持续加息，中国央行保持稳健中性，中美利差从1.5%快速收窄至0.3%警戒线附近，人民币汇率从6.2贬值至6.9
2. **2022年**：美联储大幅加息抗击通胀，中国央行稳增长宽松，中美利差首次出现长时间倒挂，人民币汇率从6.3贬值至7.3
3. **2020-2021年**：美联储零利率+无限QE，中国率先收紧货币政策，中美利差最高达到2.5%以上，外资大规模流入，人民币汇率从7.1升值至6.3
4. **2010-2013年虚假繁荣期**：中美利差扩大叠加人民币单边升值，大量套利资本通过LC融资铜等大宗商品套取息差+汇差+内外价差，形成大宗商品虚假繁荣；2013年后随美联储加息预期升温、油价崩溃、人民币贬值预期出现，利差缩小、汇差反转、投机回报率消失，三要素叠加逆转（来源：付鹏，2016-03-01）

## 局限性
- 中美利差的影响会受到资本管制政策的对冲，管制较严时利差对资本流动和汇率的影响会减弱
- 极端风险事件下，美元的避险属性会主导汇率走势，此时中美利差的影响会下降
- 利差对资产价格的影响会受到国内货币政策、经济预期等其他因素的对冲
