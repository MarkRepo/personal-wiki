## 定义
行为窗口（Behavior Window），P.E.T.中的核心分析工具。用一个长方形窗口将孩子的所有可能行为分为两部分：上方的"可接纳行为"和下方的"不可接纳行为"。分界线称为"接纳线"。

## 核心原理

### 接纳线的特性
接纳线不是一条固定的线，它会上下移动，受多种因素影响：

**父母自身因素：**
- 个人性格（有些人天生包容度强，有些人容易挑毛病）
- 当前情绪状态（精力充沛时比疲惫时更包容）
- 对行为的固有观念（什么是对的、什么是错的）

**孩子因素：**
- 不同孩子的行为特点不同（好动 vs 安静）
- 父母对不同孩子的接纳度可能不同
- 某些孩子本身就较难接纳（如高攻击性、多动）

**情境因素：**
- 时间、地点、场合
- 是否有他人在场
- 父母是否在忙

### 行为窗口的应用
行为窗口是判断"谁拥有问题"和"用什么技巧"的指南：

```
┌─────────────────┐
│   可接纳行为     │  ← 孩子行为在接纳线上方
│                 │     父母无问题，无需干预
│   ───接纳线───  │
│   不可接纳行为   │  ← 孩子行为在接纳线下方
│                 │     父母拥有问题，需要用技巧
└─────────────────┘
```

## 应用场景
- 判断当前情况下孩子行为是否真的不可接纳
- 帮助父母自我觉察：我的接纳线在哪？它合理吗？
- 区分"孩子的行为干扰了我"和"我只是觉得不应该"

## 局限性
- 接纳线的移动性使得判断有时模糊
- 父母可能对"为什么这条线会移动"缺乏自我觉察
- 需要配合"问题归属"概念使用才能发挥最大效果
