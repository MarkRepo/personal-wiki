## 概述

《大衰退：宏观经济学的圣杯》（The Holy Grail of Macroeconomics: Lessons from Japan's Great Recession）是辜朝明的代表作。书名呼应伯南克提出的"解释大萧条是宏观经济学的圣杯"，作者以日本 1990 年代大衰退为切入点，提出"资产负债表衰退"理论，试图统一凯恩斯主义与新古典主义经济学。

## 核心内容

### 全书结构

| 章节 | 主题 |
|------|------|
| 第一章 | 日本经济衰退：排除结构问题和银行问题，引入资产负债表衰退概念 |
| 第二章 | 资产负债表衰退的特征：企业融资行为、税收、利率、过早财政整顿的危害 |
| 第三章 | 美国大萧条就是一场资产负债表衰退：重新解释流动性陷阱 |
| 第四章 | 资产负债表衰退时期的金融、汇率和财政政策：批评非传统货币宽松 |
| 第五章 | 阴阳经济周期与宏观经济学的圣杯：理论框架的统一 |
| 第六章 | 全球化的压力：发达国家面临的结构转型 |
| 第七章 | 进行中的经济泡沫和资产负债表衰退：次贷危机、中国泡沫 |
| 第八章 | 陷入资产负债表衰退的世界：2008 全球金融危机的应对 |

### 核心论证逻辑

1. 日本 1990 年代衰退既非结构问题，也非银行问题（反证：低利率、外资银行无法拓展、企业债券萎缩）
2. 真正原因：资产价格暴跌 → 企业资产负债表受损 → 集体转向"负债最小化" → 合成谬误 → 总需求崩塌
3. 货币政策失效（流动性陷阱是借贷方现象，非放贷方现象）
4. 财政政策是唯一有效工具，过早财政整顿将造成灾难（桥本 1997、小泉 2001 的教训）
5. 阴阳周期理论统一凯恩斯主义和新古典主义

## 背景与历史

- 原版日文出版，英文版由 Wiley 出版
- 中文第一版 2008 年出版，书名《大衰退——如何在金融危机中幸存与发展》
- 中文修订版 2016 年出版（含第八章对 2008 全球金融危机的分析），东方出版社
- 译者：喻海翔

## 影响与评价

- 张明推荐序称本书可读性极强，分析框架对中国经济有重要参考价值
- "资产负债表衰退"概念已成为经济学界分析后泡沫经济的标准框架之一
- 2008 年全球金融危机验证了书中多项预测
- 对中国的启示：在企业去杠杆时期，必须实施积极财政政策，避免重蹈日本覆辙
